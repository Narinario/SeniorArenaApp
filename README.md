# SeniorArena
学校で開催されたハッカソン用のアプリデモ

## 技術スタック
- バックエンド: Django
- フロントエンド: React
- データベース: PostgreSQL
- コンテナ: Docker

## 環境構築手順
1. リポジトリをクローン
```sh
git clone https://github.com/TaikiHoshika/SeniorArenaApp.git
```

2. コンテナを起動
```sh
docker compose up -d
```
